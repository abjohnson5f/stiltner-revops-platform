## ⚠️ Important Notes

### Billing Unit Rounding
Google Ads requires CPC bids in USD to be multiples of 10,000 micros ($0.01). The initial script calculated:
- $1.00 × 1.15 = $1.15 (1,150,000 micros) ✅ Valid
- $1.15 × 1.15 = $1.3225 (1,322,500 micros) ❌ Invalid

The fix script rounds to nearest 10,000 micros:
- 1,322,500 → 1,320,000 micros ($1.32) ✅ Valid

This is now handled correctly in future optimizations.

### Keyword Bid Inheritance
Keywords showing "Bid: $N/A" were using the ad group default bid ($1.00). The bid updates now set explicit keyword-level bids, giving more granular control.

### Safety Measures Taken
1. **Dry run validation** before all mutations
2. **Partial failure mode** enabled (one failure doesn't stop all operations)
3. **Resource name verification** in update operations
4. **Detailed change logging** for audit trail
